标题 Tittle

Tittle: 学习 Git 和 Github 中缺失的一节课
Author: Leo Chen*
Units:

摘要 Abstract

来源: 网络及官方教程。
介绍: 分布式文件版本控制系统 Git 及依托其架构的开源代码提交平台 Github 的基础操作。
对象: Git 和 Github 的新手和学习者。
结构: 知识拆解，模块化的方式
内容: 基础知识、常见操作和易混淆部分。
讨论: Q&A

关键词 Keywords

Git、Github、教程

绪论 Introduction

预备信息: 分布式文件版本控制系统 Git、开源代码提交平台 Github。
前人工作: 各种文字和视频资源的教程。
知识空白:

学习的周期长，内容繁多。学习上，新手不能理解设计的核心，不能复用及推广。
非递进式的介绍 Git 和 Github 相关的 CLI 和 GUI 软件。使用上，新手信息闭塞，不熟悉流行趋势和易用的形式。
网络及官方教程中提及标准及模板的资源较少。规范上，新手集中以教程为模板，不知道业界的规范和模板。
动机: 自己经历了上面三个知识空白反复的迭代，希望自己整理有延申意义的学习笔记。
目的: 能设计一套适合自己学习的规范、合理、易用的学习和开发模板。
